# Kubernetes Pod Commands Cheat Sheet

## Create Pods

| Command                           | Description                         |
| --------------------------------- | ----------------------------------- |
| `kubectl run nginx --image=nginx` | Create a pod imperatively           |
| `kubectl apply -f pod.yaml`       | Create pod using YAML (declarative) |
| `kubectl create -f pod.yaml`      | Create pod from YAML file           |

---

## View Pods

| Command                      | Description                   |
| ---------------------------- | ----------------------------- |
| `kubectl get pods`           | List all pods                 |
| `kubectl get pods -o wide`   | Show pods with node and IP    |
| `kubectl get pods -A`        | List pods in all namespaces   |
| `kubectl get pod <pod-name>` | Get details of a specific pod |

---

## Describe and Inspect

| Command                              | Description                    |
| ------------------------------------ | ------------------------------ |
| `kubectl describe pod <pod-name>`    | Detailed information about pod |
| `kubectl get pod <pod-name> -o yaml` | View pod configuration in YAML |
| `kubectl get pod <pod-name> -o json` | View pod configuration in JSON |

---

## Logs

| Command                                       | Description                  |
| --------------------------------------------- | ---------------------------- |
| `kubectl logs <pod-name>`                     | View pod logs                |
| `kubectl logs -f <pod-name>`                  | Stream logs (follow)         |
| `kubectl logs <pod-name> -c <container-name>` | Logs from specific container |

---

## Execute Commands Inside Pod

| Command                               | Description                      |
| ------------------------------------- | -------------------------------- |
| `kubectl exec <pod-name> -- ls`       | Run command inside pod           |
| `kubectl exec -it <pod-name> -- bash` | Interactive shell inside pod     |
| `kubectl exec -it <pod-name> -- sh`   | Open shell if bash not available |

---

## Port Forwarding

| Command                                       | Description               |
| --------------------------------------------- | ------------------------- |
| `kubectl port-forward pod/<pod-name> 8080:80` | Forward local port to pod |

---

## Delete Pods

| Command                         | Description                   |
| ------------------------------- | ----------------------------- |
| `kubectl delete pod <pod-name>` | Delete a specific pod         |
| `kubectl delete -f pod.yaml`    | Delete pod created using YAML |
| `kubectl delete pods --all`     | Delete all pods in namespace  |

---

## Debugging Pods

| Command                           | Description             |
| --------------------------------- | ----------------------- |
| `kubectl describe pod <pod-name>` | Check events and errors |
| `kubectl get events`              | View cluster events     |
| `kubectl logs <pod-name>`         | Check application logs  |

---

## Labels and Selectors

| Command                                 | Description         |
| --------------------------------------- | ------------------- |
| `kubectl label pod <pod-name> env=prod` | Add label to pod    |
| `kubectl get pods -l env=prod`          | Get pods with label |

---

## Pod Status Monitoring

| Command                                                  | Description                    |
| -------------------------------------------------------- | ------------------------------ |
| `kubectl get pods -w`                                    | Watch pod changes in real time |
| `kubectl get pods --field-selector=status.phase=Running` | Filter running pods            |

---

# Quick Debug Flow

```bash
kubectl get pods
kubectl describe pod <pod-name>
kubectl logs <pod-name>
kubectl exec -it <pod-name> -- sh
```
